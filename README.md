<b>Problem Statement</b>

Residential societies often manage maintenance payments, complaints, and expenses manually using spreadsheets and paperwork.
This leads to:

Payment tracking issues

Lack of transparency

Poor communication between residents and management

MaintenancePro solves this by providing a centralized digital platform for society management.

<b>Installation & Setup</b>

<b>Clone repository</b>

git clone https://github.com/yourusername/maintenance.git

<b>Frontend setup</b>

cd frontend

npm install

npm run dev

<b>Backend setup</b>
cd backend
npm install
npm run start

Create .env file in backend:

MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
